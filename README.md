# voting-session

Application to open a session and insert topics in it. The users can vote during the \
session time
duration. When session ends, result is send by message.

## Running application

1° step -> build gradle \
2° step -> docker-compose up

## Swagger
http://localhost:8080/swagger-ui.html

## Stack
Java 11 \
MongoDB \
WebFlux \
Kafka \
Docker
